**Frontend:**
- Vue.js 3 + TypeScript
- Tailwind CSS
- Axios for API calls

**Backend:**
- FastAPI
- Python

### Prerequisites
- Python 3.7+
- pnpm

### Frontend Setup
```bash
cd frontend
pnpm install
pnpm run dev
```

### Backend
```
cd backend
pip install "fastapi[standard]"
pip install pythainlp
```

**Start Frontend:**
```bash
cd frontend && pnpm run dev
```

**Start Backend:**
```bash
cd backend && fastapi dev main.py
```
